활성 엔드포인트

FortiGate / FortiManager 연동
연동 가이드: FortiGate/FortiManager에서 External Block List 또는 Threat Feed로 등록하여 사용할 수 있습니다. 각 엔드포인트는 실시간으로 활성 IP를 제공하며, 화이트리스트는 자동으로 제외됩니다.
GET 활성 IP 목록 (JSON)

페이지네이션 지원, 상세 메타데이터 포함 (국가, 신뢰도, 탐지일 등)

FortiGate 설정 방법:
Security Fabric > External Connectors > Create New > Threat Feeds > URL: [이 URL 입력] > Update Interval: Hourly
GET Threat Feed (Push API 형식)

FortiGate 7.2+ Push API 호환 JSON 형식 (snapshot/add/remove 명령 지원)

FortiManager 자동화 스크립트
FortiManager 통합: 서버의 /scripts 디렉토리에 FortiManager CLI Script 등록 및 자동 업로드 스크립트가 준비되어 있습니다.
CLI Script 등록

FortiManager에 주기적 실행 스크립트 등록

bash /scripts/fmg-register-script.sh \
  [FMG_IP] [USER] [PASSWORD]
자동 업로드 스크립트

Flask API → FortiManager → FortiGates

bash /scripts/fmg-auto-upload.sh \
  [FMG_IP] [USER] [PASSWORD]
Cron 설정 예시 (5분마다 업데이트):
*/5 * * * * bash /scripts/fmg-auto-upload.sh 192.168.1.100 admin 'password' >> /var/log/fmg-upload.log 2>&1
