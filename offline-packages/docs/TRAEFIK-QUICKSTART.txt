📦 파일: traefik-offline.tar.gz (48M)

═══════════════════════════════════════════════════════════════════

🚀 설치 (3단계)

1️⃣  압축 해제
    tar -xzf traefik-offline.tar.gz
    cd traefik-offline/

2️⃣  자동 설치
    bash install-traefik.sh

3️⃣  확인
    docker ps | grep traefik
    curl http://localhost:8080/api/http/routers

═══════════════════════════════════════════════════════════════════

✅ 접속 정보

• 대시보드:  http://서버IP:8080
• HTTP:      http://서버IP:80
• HTTPS:     https://서버IP:443

═══════════════════════════════════════════════════════════════════

🔧 Blacklist 연동 (추가 설치)

tar -xzf blacklist.tar.gz
cd blacklist/
bash install.sh

→ Traefik가 자동으로 blacklist.nxtd.co.kr 라우팅

═══════════════════════════════════════════════════════════════════

❓ 문제 해결

• 로그 확인:      docker logs traefik
• 포트 확인:      ss -tuln | grep -E ':(80|443|8080)'
• 재시작:         docker restart traefik
